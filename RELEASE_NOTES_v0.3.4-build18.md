# LocalShot 0.3.4（Build 18）公开测试版

本版本修复录屏已经保存到统一目录，但没有进入“最近保存”，且“清空记录”确认弹窗仍只描述截图的问题。

## 最近保存包含录屏

- “最近保存”现在按时间同时显示截图和录屏。
- 截图继续打开快速访问；录屏点击后在 Finder 中定位。
- 录屏完成后立即刷新菜单，不需要重启 LocalShot。
- 只有录屏、没有截图时，“清空记录”按钮也可以正常使用。

## 清空记录逻辑

- 确认弹窗会同时说明截图、录屏和标注项目。
- 不勾选删除文件时，只隐藏截图和录屏记录，原文件继续保留。
- 勾选“同时删除截图和录屏文件”后，才删除当前目录内由 LocalShot 生成的截图、对应标注项目和最终 MP4。
- 个人 MP4、临时录制分段和旧版独立 `Application Support/LocalShot/Recordings` 目录不会被删除。

## 兼容与隐私

- 同一 Universal 2 安装包支持 Apple 芯片（arm64）和 Intel（x86_64）Mac，最低 macOS 13。
- 区域屏幕录制和原位置翻译仍需要 macOS 15 或更高版本。
- 所有索引和删除判断均在本机完成，不增加网络请求、账号或云服务。
- 覆盖安装不会主动删除截图、录屏、设置或离线授权；macOS 可能因应用签名变化要求重新允许屏幕捕获。

- 请勿从第三方转载链接下载安装包。
