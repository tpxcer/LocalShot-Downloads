# LocalShot 0.3.3（Build 17）公开测试版

本版本合并截图与录屏的菜单入口和本地保存目录，让功能入口更紧凑，也便于在一个 Finder 目录中统一查找新截图与新录屏。

## 菜单调整

- 状态栏菜单的“截图”分组改为“功能”。
- “区域录屏”移动到“功能”分组，录屏能力保持不变。
- 取消独立“录制”分组和“打开录制目录”入口。
- “最近截图”分组改为“最近保存”。
- “打开截图目录 / 清空截图记录”简化为“打开目录 / 清空记录”。
- 使用“打开目录”统一查看截图与新录屏。

## 统一本地目录

- 新录屏与截图默认保存在 `Application Support/LocalShot/Captures`。
- 在设置中选择自定义目录后，截图和新录屏会一起跟随该目录。
- MP4 继续使用 `-recording.mp4` 文件名，不进入截图历史，也不会被“清空记录”删除。
- 旧版本 `Application Support/LocalShot/Recordings` 中的文件不会自动移动或删除。

## 兼容与隐私

- 同一 Universal 2 安装包支持 Apple 芯片（arm64）和 Intel（x86_64）Mac，最低 macOS 13。
- 区域屏幕录制和原位置翻译仍需要 macOS 15 或更高版本。
- 所有截图、录屏和目录操作均在本机完成，不增加网络请求、账号或云服务。
- 覆盖安装不会主动删除截图、录屏、设置或离线授权；macOS 可能因应用签名变化要求重新允许屏幕捕获。

- 当前版本未经 Apple 公证，macOS 首次启动时可能需要在“系统设置 → 隐私与安全性”中点击“仍要打开”。
- 请勿从第三方转载链接下载安装包。
