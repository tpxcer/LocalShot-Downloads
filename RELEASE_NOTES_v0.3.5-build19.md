# LocalShot 0.3.5（Build 19）公开测试版

本版本按用户反馈精简状态栏“功能”分组，减少重复入口和过长文字。

## 状态栏菜单

- 状态栏不再显示“区域长截图”。
- 区域长截图功能继续保留，可从截图选区工具栏进入。
- “识别屏幕文字”简化为“识别文字”。
- 区域、窗口、全屏截图和区域录屏入口保持不变。
- 全局识别文字快捷键仍为 `⌘⇧O`。

## 兼容与隐私

- 同一 Universal 2 安装包支持 Apple 芯片（arm64）和 Intel（x86_64）Mac，最低 macOS 13。
- 区域屏幕录制和原位置翻译仍需要 macOS 15 或更高版本。
- 不改变截图、长截图、录屏、最近保存或清空记录的数据逻辑。
- 覆盖安装不会主动删除截图、录屏、设置或离线授权；macOS 可能因应用签名变化要求重新允许屏幕捕获。

## 下载文件

- `LocalShot-0.3.5-build19-offline-license-universal2-arm64-x86_64.dmg`：Apple 芯片与 Intel Mac 通用客户安装包。
- `SHA256SUMS-v0.3.5-build19-public.txt`：安装包完整性校验值。

## 重要提示

- 当前版本未经 Apple 公证，macOS 首次启动时可能需要在“系统设置 → 隐私与安全性”中点击“仍要打开”。
- 请勿从第三方转载链接下载安装包。
- 安装、升级或激活方法见仓库中的 `INSTALL.md`。
