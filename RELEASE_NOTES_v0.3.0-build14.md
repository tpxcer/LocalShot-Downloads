# LocalShot 0.3.0（Build 14）公开测试版

本版本集中完善区域屏幕录制的稳定性、开始提示、暂停控制、画质选择与存储安全，仅支持 Apple 芯片 Mac。

## 录制稳定性

- 录制期间阻止显示器和系统自动睡眠，避免长时间录制被意外中断。
- 系统睡眠或用户会话切换前会自动停止，并尽量保存已经录制的有效内容。
- 锁定窗口短暂隐藏、最小化或切换空间时，使用 ScreenCaptureKit 二次确认，不再立即误判为窗口关闭。
- 显示器配置变化后会重新定位红色范围框和悬浮控制条。
- 锁定窗口关闭时继续自动停止并保存；手动区域仍固定在原屏幕位置。

## 倒计时、暂停与全局停止

- 控制条明确显示“锁定窗口”或“固定区域”。
- 点击开始后显示 3、2、1 倒计时；倒计时期间可以取消，不会提前写入视频。
- 录制中新增暂停/继续。暂停期间计时和视频时间轴都停止，继续后最终仍只生成一个 MP4。
- 录制期间可在任意应用中按 `⌘⇧R` 停止；录制结束后快捷键立即释放，不长期占用。

## 画质、帧率与磁盘安全

- 新增标准（最长边 1280）、高清（最长边 1920）和原画三档画质。
- 新增 30 FPS 与 60 FPS，可在设置或开始录制前的控制条选择。
- 控制条显示每分钟文件大小估算范围和录制目录所在磁盘的可用空间。
- 磁盘空间偏低时提醒；达到安全线时自动停止并保存，避免因磁盘写满破坏整段录制。

## 本地与隐私边界

- 截图、录屏、系统声音和可选麦克风声音仍只写入用户 Mac 本地。
- 不增加账号、云同步、广告、分析统计或录屏上传服务。
- 录屏临时分段只用于本机暂停/继续合并，成功后自动清理；合并失败时保留已有分段，避免无提示丢失内容。

## 系统要求与升级

- Apple 芯片 Mac，最低 macOS 13；区域屏幕录制和原位置翻译需要 macOS 15 或更高版本。
- Build 5–13 用户可通过软件内更新提示下载 Build 14；Build 4 及更早版本需要手动下载。
- 覆盖安装不会主动删除截图、录屏、设置或离线授权。如果 macOS 因应用签名身份变化再次询问屏幕捕获权限，请按系统提示重新允许。
