# LocalShot 安装说明

## 系统要求

- Apple 芯片 Mac（M1、M2、M3、M4 或后续 arm64 机型）。
- macOS 13 或更高版本。

## 安装

1. 从本仓库的 Releases 下载 `LocalShot-0.1.1-build4-offline-license-arm64.dmg`。
2. 双击打开 DMG。
3. 将 `LocalShot.app` 拖入 `Applications`（应用程序）文件夹。
4. 在应用程序文件夹中打开 LocalShot。

## 首次打开提示

当前测试版未经 Apple 公证。如果 macOS 提示“无法验证开发者”或阻止打开：

1. 打开“系统设置”。
2. 进入“隐私与安全性”。
3. 在安全性提示旁点击“仍要打开”。
4. 按照 macOS 提示确认。

请只从本仓库 Release 下载，并可使用发布附件中的 SHA-256 文件核对安装包。

## 屏幕捕获权限

LocalShot 第一次截图时需要 macOS 屏幕捕获权限。请按软件提示前往系统设置允许 LocalShot，然后重新打开软件。

LocalShot 不需要辅助功能权限即可完成当前手动滚动长截图。

## 激活

1. 打开 LocalShot 设置中的“软件授权”。
2. 点击复制机器码。
3. 把机器码发送给向你提供软件的销售方。
4. 收到授权码后粘贴到软件中并点击激活。

授权验证完全在本机进行，不需要登录账号或连接授权服务器。
