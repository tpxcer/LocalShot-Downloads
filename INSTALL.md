# LocalShot 安装说明

## 系统要求

- Apple 芯片 Mac（arm64）或 Intel Mac（x86_64）。
- macOS 13 或更高版本。

## 安装

1. 从本仓库的 Releases 下载 `LocalShot-0.3.2-build16-offline-license-universal2-arm64-x86_64.dmg`。
2. 双击打开 DMG。
3. 将 `LocalShot.app` 拖入 `Applications`（应用程序）文件夹。
4. 在应用程序文件夹中打开 LocalShot。

## 首次打开提示

当前测试版未经 Apple 公证。如果 macOS 提示“无法验证开发者”或阻止打开：

1. 打开“系统设置”。
2. 进入“隐私与安全性”。
3. 在安全性提示旁点击“仍要打开”。
4. 按照 macOS 提示确认。

请只从本仓库 Release 下载，并可使用发布附件中的 SHA-256 文件核对安装包。

## 屏幕捕获权限

LocalShot 第一次截图或区域录屏时需要 macOS 屏幕捕获权限。请按软件提示前往系统设置允许 LocalShot，然后重新打开软件。

区域录屏需要 macOS 15 或更高版本。系统声音可直接在录屏控制条选择；麦克风默认关闭，只有主动开启时才会请求麦克风权限。

LocalShot 不需要辅助功能权限即可完成当前手动滚动长截图。

## 激活

1. 打开 LocalShot 设置中的“软件授权”。
2. 点击复制机器码。
3. 把机器码发送给向你提供软件的销售方。
4. 收到授权码后粘贴到软件中并点击激活。

授权验证完全在本机进行，不需要登录账号或连接授权服务器。

## 升级到 Build 16

Build 5–15 用户可以在软件内收到 Build 16 提示并下载 DMG。Build 4 及更早版本没有更新检查功能，需要从本仓库手动下载 Build 16 DMG。打开后把新版 `LocalShot.app` 拖入“应用程序”覆盖旧版。

截图历史、设置和离线授权保存在应用包之外，LocalShot 覆盖安装不会主动删除这些数据。如果 macOS 因应用签名身份变化再次询问屏幕捕获权限，请按系统提示重新允许。

## 后续版本更新

从 Build 5 开始，LocalShot 默认每 24 小时最多检查一次公开 GitHub Release，并在发现新版时提示：

1. 点击提示中的“下载更新”，或在设置的“关于”中点击“检查更新”。
2. LocalShot 下载 DMG，并核对文件大小和 GitHub SHA-256 digest。
3. 校验通过后，DMG 会保存到“下载”文件夹并自动打开。
4. 将新版 `LocalShot.app` 拖入“应用程序”，选择覆盖旧版，再重新打开 LocalShot。

可以在设置中关闭“自动检查更新”；手动检查按钮仍然可用。LocalShot 不会在后台静默下载或替换应用。
