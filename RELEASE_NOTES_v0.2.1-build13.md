# LocalShot 0.2.1（Build 13）公开测试版

本版本重点优化区域屏幕录制的选窗和跟随体验，仅支持 Apple 芯片 Mac。

## 锁定窗口录制

- 区域录屏启动后会自动识别鼠标所在窗口。
- 单击高亮窗口会锁定该窗口；移动窗口后，录制内容、红色范围框和悬浮控制条同步跟随。
- 调整锁定窗口大小时，红框同步变化，录制画面等比例适配到当前 MP4 的固定视频尺寸。
- 手动拖动选择区域时，继续录制固定屏幕位置，不会跟随下方窗口。
- 目标窗口在准备阶段关闭会自动取消；录制期间关闭时会尝试停止并保存有效视频，并显示结果提示。

## 录屏界面优化

- 红色录制范围框加粗，并在准备、启动、录制和保存阶段持续位于最上层。
- 窗口识别支持悬停高亮、窗口名称与尺寸提示；单击选窗和拖动自由选区使用同一个入口。
- LocalShot 的红框与悬浮控制条继续排除在录制成片之外。

## 隐私边界

- 截图、录屏、系统声音和可选麦克风声音都只写入用户 Mac 本地。
- 不增加账号、云同步、广告、分析统计或录屏上传服务。
- 直接分发版仍只有更新检查与用户确认后的 DMG 下载会访问公开 GitHub Release。

## 系统要求与升级

- Apple 芯片 Mac，最低 macOS 13；区域屏幕录制和原位置翻译需要 macOS 15 或更高版本。
- Build 5–12 用户可通过软件内更新提示下载 Build 13；Build 4 及更早版本需要手动下载。
- 覆盖安装不会主动删除截图、录屏、设置或离线授权。如果 macOS 因应用签名身份变化再次询问屏幕捕获权限，请按系统提示重新允许。
