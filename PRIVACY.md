# LocalShot 隐私说明

更新日期：2026 年 7 月 24 日

LocalShot 是纯本地 macOS 截图与区域录屏工具。当前版本不提供账号、云同步、团队协作、广告或在线存储服务。

## 数据处理

- LocalShot 不收集或上传截图、区域录屏、录制声音、导入图片、OCR 文字、翻译结果、标注内容或历史记录。
- LocalShot 不包含分析统计、用户行为追踪、广告标识符或远程崩溃日志上传。
- OCR 使用 macOS Vision 在本机执行。
- 原位置翻译使用 macOS Apple Translation。翻译所需语言资源由 macOS 管理，系统可能联网下载语言资源；LocalShot 不会把截图或 OCR 文字上传到自己的服务器。
- 直接分发版本默认每 24 小时向公开 GitHub Release API 查询一次版本信息，可在设置中关闭，也可以手动检查。

## 离线授权

LocalShot 在本机读取系统硬件标识，并只在本机生成不可逆的哈希机器码，用于验证离线授权码。应用不会上传机器码、授权码或原始硬件标识。只有用户主动复制并发送机器码时，销售方才会收到该机器码。

## 本地数据

- 截图和新录制的区域录屏默认保存在本机 `Application Support/LocalShot/Captures`，也可以在设置中为两者选择同一个其他本地目录。
- 旧版本保存在 `Application Support/LocalShot/Recordings` 的录屏不会自动移动或删除。
- 历史 OCR 搜索索引、标注项目和设置保存在本机。
- 只有用户主动选择复制时，图片或文字才会写入系统剪贴板。
- 只有用户主动选择导出或另存为时，文件才会写入用户选择的位置。
- 用户可以删除单张截图或主动清空 LocalShot 管理的保存记录。
- 区域录屏不进入截图缩略图历史，但会显示在“最近保存”中。
- “清空记录”不勾选删除文件时只隐藏截图和录屏记录；只有用户主动勾选后，当前目录内由 LocalShot 生成的截图、标注项目和最终 MP4 才会删除。
- 个人视频、临时录制分段和旧版独立 `Recordings` 目录不会被“清空记录”删除。
- 导入的外部原图片不会被 LocalShot 删除。

## 系统权限

- 屏幕与系统音频捕获权限用于用户主动发起的区域、窗口、全屏、滚动截图和区域屏幕录制。
- 麦克风权限只在用户主动开启录屏麦克风时请求；默认关闭时不会采集麦克风声音。
- 文件访问通过 macOS 文件选择器完成。
- 登录项只会在用户主动开启后注册。

LocalShot 不会绕过系统权限，也不会在用户未操作时自行开启权限或登录项。

## 网络与第三方

LocalShot 不运营云服务。版本检查会访问 `api.github.com` 查询本仓库的公开 Release：

- 请求不包含截图、OCR 文字、机器码、授权码、历史、文件路径或使用记录。
- GitHub 可能看到常规网络 IP、请求时间和固定的更新检查 User-Agent，这些数据由 GitHub 按其隐私政策处理。
- 只有用户确认后才下载公开 DMG；文件通过 GitHub SHA-256 digest 校验后保存到“下载”文件夹并打开。
- 关闭“自动检查更新”后不会后台请求 GitHub；手动检查或下载仍会在用户点击时联网。

macOS 可能为系统翻译语言资源、系统更新或其他操作系统功能访问 Apple 服务，这些行为由 macOS 控制并受 Apple 隐私政策约束。

如有隐私问题，请在公开下载仓库提交 Issue，但不要附带机器码、授权码或私人截图。
