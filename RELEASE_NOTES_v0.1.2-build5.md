# LocalShot 0.1.2（Build 5）公开测试版

这是 LocalShot 的 Build 5 更新版本，仅支持 Apple 芯片 Mac。

包含区域、窗口、全屏和手动区域长截图，以及标注、本地 OCR、系统翻译、截图置顶、图片美化、本地历史和离线永久授权。

## 本次新增：软件更新提醒

- 默认每 24 小时最多检查一次公开 GitHub Release，并在发现新版时提示。
- 设置页新增“自动检查更新”开关、“检查更新”按钮和当前检查状态。
- 用户确认后才下载新版 DMG，不会在后台静默下载或替换应用。
- 下载完成后核对 GitHub 提供的文件大小和 SHA-256 digest；校验失败不会打开文件。
- 校验通过后把 DMG 保存到用户“下载”文件夹并自动打开，再由用户拖入“应用程序”覆盖旧版。
- 自动检查可关闭；关闭后截图、授权和其他核心功能仍可离线使用。

## 从旧版本升级

Build 4 及更早版本没有更新检查代码，因此需要最后手动下载并安装一次 Build 5。从 Build 5 开始，后续版本可在软件内收到提示。

覆盖安装不会主动删除截图历史、设置或离线授权。当前安装包仍未经 Apple 公证，macOS 也可能要求重新确认屏幕捕获权限。

## 隐私

更新检查只查询公开 GitHub Release，不发送截图、OCR、机器码、授权码、历史、文件路径或使用记录。GitHub 可能看到常规 IP、请求时间和固定 User-Agent，详细说明见 `PRIVACY.md`。

## 下载文件

- `LocalShot-0.1.2-build5-offline-license-arm64.dmg`：客户安装包。
- `SHA256SUMS-v0.1.2-build5-public.txt`：安装包完整性校验值。

## 重要提示

- 当前版本未经 Apple 公证，macOS 首次启动时可能需要在“系统设置 → 隐私与安全性”中点击“仍要打开”。
- 请勿从第三方转载链接下载安装包。
- 安装、升级或激活方法见仓库中的 `INSTALL.md`。
