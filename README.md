# LocalShot 下载

LocalShot 是一款纯本地 macOS 截图工具，不包含视频录制、账号、云同步、广告或在线存储。

## 下载

请从 [当前公开测试版](https://github.com/tpxcer/LocalShot-Downloads/releases/tag/v0.1.1-build4) 下载 DMG。

当前公开测试版：`0.1.1 (Build 4)`

- 支持 Apple 芯片 Mac（arm64）。
- 最低系统版本为 macOS 13。
- 原位置系统翻译需要 macOS 15 或更高版本，以及 macOS 提供的对应语言资源。
- 当前版本未经 Apple 公证，首次启动请按照 [安装说明](INSTALL.md) 操作。

## 主要功能

- 区域、窗口和全屏截图。
- 鼠标或触控板手动滚动的区域长截图，并实时显示拼接预览。
- 箭头、画笔、马赛克、模糊、椭圆、矩形、步骤编号和文字标注。
- 本地 OCR、系统翻译、截图置顶和图片美化。
- 本地截图历史与文字搜索。
- PNG/JPEG 输出、自定义快捷键和保存目录。

## 离线授权

LocalShot 使用一机一码永久离线授权。安装后在软件设置中复制机器码，发送给向你提供软件的销售方，再把收到的授权码粘贴回软件即可。应用不会自动上传机器码或授权码。

授权码与生成时的 Mac 绑定，不能在另一台机器上直接使用。

## 隐私

截图、OCR、标注、历史和授权验证均在本机完成。详细说明见 [隐私说明](PRIVACY.md)。
