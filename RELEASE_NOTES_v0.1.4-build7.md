# LocalShot 0.1.4（Build 7）公开测试版

这是 LocalShot 的 Build 7 截图定位体验调整，仅支持 Apple 芯片 Mac。

## 本次调整

- 彩虹截图指针由 30pt 缩回 macOS 默认箭头的标准视觉尺寸，只保留彩虹配色、高对比轮廓和左上尖端热点。
- 像素放大镜由圆形改为直角正方形，显示尺寸从 144×144pt 扩大到 168×168pt。
- 放大镜采样范围从 48×48px 扩大到 56×56px，显示与采样同步扩大并保持原有 3× 像素倍率。
- 放大镜靠近截图边缘时固定使用正方形采样框，不再因裁窄采样图而拉伸内容。
- 主快捷截图浮层和兼容区域选择浮层使用同一套尺寸与方形样式。

## 从旧版本升级

Build 5/6 用户可通过软件内更新提示下载 Build 7。Build 4 及更早版本需要从本仓库手动下载 DMG。

覆盖安装不会主动删除截图历史、设置或离线授权。当前安装包仍未经 Apple 公证，macOS 也可能要求重新确认屏幕捕获权限。

## 下载文件

- `LocalShot-0.1.4-build7-offline-license-arm64.dmg`：客户安装包。
- `SHA256SUMS-v0.1.4-build7-public.txt`：安装包完整性校验值。

## 重要提示

- 当前版本未经 Apple 公证，macOS 首次启动时可能需要在“系统设置 → 隐私与安全性”中点击“仍要打开”。
- 请勿从第三方转载链接下载安装包。
- 安装、升级或激活方法见仓库中的 `INSTALL.md`。
