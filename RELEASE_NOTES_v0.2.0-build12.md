# LocalShot 0.2.0（Build 12）公开测试版

这是 LocalShot 首个支持区域屏幕录制的公开测试版，仅支持 Apple 芯片 Mac。

## 新增区域屏幕录制

- macOS 15 或更高版本可从菜单选择“区域录屏”，也可在截图选区工具栏点击圆形录制图标。
- 录制前自由框选区域，准备和录制期间持续显示红色细边框。
- 悬浮控制条提供红色状态点、`00:00:00` 计时、系统声音、麦克风、取消和开始/停止按钮。
- 系统声音默认开启，麦克风默认关闭；只有用户主动开启麦克风时才请求系统权限。
- 使用 macOS ScreenCaptureKit 输出本地 H.264 MP4，完成后自动在 Finder 中定位文件。
- 录制文件独立保存在 `Application Support/LocalShot/Recordings`，不进入截图历史，也不会被“清空截图记录”删除。

## 隐私边界

- 截图、录屏、系统声音和可选麦克风声音都只写入用户 Mac 本地。
- 不增加账号、云同步、广告、分析统计或录屏上传服务。
- 直接分发版仍只有更新检查与用户确认后的 DMG 下载会访问公开 GitHub Release。

## 系统要求

- Apple 芯片 Mac，最低 macOS 13。
- 区域屏幕录制和原位置翻译需要 macOS 15 或更高版本。
- 首次录屏需要按 macOS 提示允许屏幕捕获；麦克风权限仅在用户主动开启时需要。

## 从旧版本升级

Build 5–11 用户可通过软件内更新提示下载 Build 12。Build 4 及更早版本需要从本仓库手动下载 DMG。

覆盖安装不会主动删除截图、录屏、设置或离线授权。如果 macOS 因应用签名身份变化再次询问屏幕捕获权限，请按系统提示重新允许。

## 下载文件

- `SHA256SUMS-v0.2.0-build12-public.txt`：安装包完整性校验值。

## 重要提示

- 当前版本未经 Apple 公证，macOS 首次启动时可能需要在“系统设置 → 隐私与安全性”中点击“仍要打开”。
- 请勿从第三方转载链接下载安装包。
- 安装、升级或激活方法见仓库中的 `INSTALL.md`。
